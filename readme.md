1. **Заклоньте проект с моего репозитория:**
    ```bash
        git clone <ссылка на репозиторий>
    ```
2. **Создайте env и пропишите название бд:**
    DB_NAME=<название бд>

3. **Создайте суперюзера:**
     ```bash
        ./manage.py createsuperuser
    ```

4. **Запустите проект:**
      ```bash
        ./manage.py runserver
    ```
